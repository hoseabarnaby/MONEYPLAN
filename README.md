# MoneyPlan v14 - Mobile colorful sections

Versi ini tetap single HTML, tetapi desain mobile dibuat lebih jelas:

- Tema terang
- Section punya garis warna di sisi kiri
- Kartu statistik punya warna berbeda

- Tetap ada Download PDF dan Export CSV

Isi Supabase di `index.html`, lalu jalankan via localhost.
